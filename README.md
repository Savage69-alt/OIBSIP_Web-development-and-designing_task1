# T-Shirt Landing Page Website

## Internship Task
OIBSIP Web Development – Task 1

## Objective
To design and develop a responsive and visually appealing landing page for a T-shirt brand using basic HTML and CSS. The goal of this project is to understand layout structuring, styling, alignment, and UI design fundamentals.

## Technologies Used
- HTML5
- CSS3
- Visual Studio Code

## Features
- Clean and modern landing page layout
- Hero section with product image and brand tagline
- Well-aligned sections with proper padding and spacing
- Call-to-action button
- Header and footer sections
- Image integration using CSS and HTML
- Beginner-friendly and readable code structure

## Project Structure
│── index.html
│── style.css
│── images/
│ └── hero.jpg
│── README.md

## How to Run the Project
1. Download or clone the repository
2. Open the project folder in Visual Studio Code
3. Open `index.html` in any web browser  
   **OR**
4. Use Live Server extension in VS Code for better experience

## Outcome
Successfully created a beginner-level T-shirt landing page demonstrating foundational knowledge of HTML and CSS, including layout design, image handling, and responsive styling.
